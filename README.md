# 💬 HelloTalk
A full-stack real-time chat application built with **Spring Boot**, **React.js**, **MySQL**, and **WebSocket (STOMP)**. The platform provides secure user authentication, friend management, presence tracking, and instant messaging with a responsive user interface.
## 🚀 Features

### Authentication
* User Registration
* Email OTP Verification
* JWT Authentication
* Secure Login & Logout
* Password Encryption using BCrypt

### Real-Time Communication
* One-to-One Messaging
* WebSocket + STOMP Integration
* Live Message Delivery
* Online/Offline Presence Tracking
* Real-Time Updates

### Social Features
* Search Users
* Send Friend Requests
* Accept Friend Requests
* Reject Friend Requests
* Friends List Management
* User Profiles

### Admin Panel
* User Monitoring
* User Management
* Administrative Controls
* Dashboard Access
